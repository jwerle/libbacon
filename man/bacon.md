bacon(1) -- Command line Baconian Cipher utility
=================================

## SYNOPSIS

`bacon` [-hV] <command> [alpha]

## OPTIONS

  -V, --version
      output program version

  -h, --help
      output help information

## INPUT

  [alpha]
      cipher alphabet. Default: 'ABCDEFGHIKLMNOPQRSTUWXYZ'

## COMMANDS

  `encode`
      Encode stdin stream

  `decode`
      Decode stdin stream
